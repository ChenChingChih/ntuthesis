\textbf{中文} 中文 \emph{中文} By default, bookdown merges all Rmd files
by the order of filenames, e.g., \texttt{01-intro.Rmd} will appear
before \texttt{02-literature.Rmd}. Filenames that start with an
underscore \texttt{\_} are skipped. \texttt{index.Rmd} will always be
treated as the first file.

Alternatively, set the order in \texttt{\_bookdown.yml}:

\begin{Highlighting}[]
\FunctionTok{rmd_files:}

  \FunctionTok{latex:}\AttributeTok{ }\KeywordTok{[}\StringTok{"abstract.Rmd"}\KeywordTok{,} \StringTok{"intro.Rmd"}\KeywordTok{]}
\end{Highlighting}
\end{Shaded}
