By default, bookdown merges all Rmd files by the order of filenames, e.g., 01-intro.Rmd will appear before 02-literature.Rmd. Filenames that start with an underscore _ are skipped. index.Rmd will always be treated as the first file.

Stanford (2008) 英文參考文獻。中文參考文獻 (黃宣範, 1993)

Alternatively, set the order in _bookdown.yml:

rmd_files:
  html: ["index.Rmd", "abstract.Rmd", "intro.Rmd"]
  latex: ["abstract.Rmd", "intro.Rmd"]
